# Tetris

Classic Tetris game with 7-bag randomization, ghost piece, next piece preview, scoring, and level progression.

**Built by:** [Sentient Brewer](https://sentientbrewer.com)
**Platform:** Windows (.exe) via Python + pygame
**Controls:** Left/Right arrows to move, Up to rotate, Down for soft drop, Space for hard drop, ESC to pause, R to restart
